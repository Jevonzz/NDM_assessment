## Getting Started

### Installing

```bash
# Clone the repo
git clone https://github.com/Jevonzz/NDM_assessment.git

# Navigate to the project directory
cd nine-dynasty-marketing-assessment

# Install dependencies
npm install

# Start the development server
npm run dev
```
